    MaxMSP

    MaxMSP is a computer programming language for sound and visual. Most of my electroacoustic sets are performed with MaxMSP.

    My Max patch research includes:

    • SP-Tools, machine learning tools for real-time improvisation developed by Rodrigo Constanzo using FluCoMa Toolkit
    • ambisonic sound spatialization by ICST Institute for Computer Music and Sound Technology, which I play in quadraphonic set up if being supported by venues
    • mc. objects, which I can use to distribute single sound into multiple pitches and speeds at once
    • OSC data and Max communication
    • RNBO & Raspberry Pi for sound installation

    I'm interested in giving workshops/lectures/exchanges/residencies about programming in Max, spatial sound improvisation, arduino-Max incoporation, MIDI-Max control, and more. Please reach out to me. I'm happy to share and learn together!

    ORCA

    I taught the Livecoding Sound class at RISD in winter 2025. This class emphasizes on using code as material to program algorithmic sequences to create generative pattern for sound. No prior coding experience or musical background is required (even better) for this class.

    class video

    Topics we discussed in class including Computer Music History, Algorave Culture, Code as Mudium, Algorithmic and Generative Music, More-than-western rhythm and sequence, Sound Improvisation. Class structured around Workshop, Jam session (Jamuary), In-class Listening session, Reading Discussion<, Visiting Artist Lecture/Workshop, Final Performance.

    Software we learned:

    • Hundred Rabbits ORCA & Pilot
    • VCV Rack
    • Ableton Live
    • MIDI and UDP communication
    Class materials:
    • Orca x Pilot Cheat Sheet
      • (including shortcut for Orca, Pilot, and ready-to-use coding modules)
    • Miro Board
      • (for commnunity building, knowledge sharing, and media responding)
    • Live Coding: A User's Manual
      • (for understanding the context, history, community, and medium of livecoding)

    This class invited Femi Shunoga-Fleming as the visiting artist for this class. Femi's live coding practice is inspried by nature, making his sound generative and textural that grows from the digital root.

    Teaching Journal

    This class included students from jewelry, textile, architecture, illustration, and graphic design. I was amazed by their intuition and the way they incorporated their professional backgrounds into code and sound. I will never forget the day one of my students told me he felt the sound he created both was and was not music. He said it left him confused, because he couldn’t place himself within any defined realm of music. I was so happy for him to make this discovery and told him that being flexible and confident in the in-between is one of the most powerful spaces to be.
