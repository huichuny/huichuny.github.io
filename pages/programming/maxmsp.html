    MaxMSP is a computer programming language for sound and visual. Most of my electroacoustic sets are performed with MaxMSP.

    My Max patch research includes:

    • SP-Tools, machine learning tools for real-time improvisation developed by Rodrigo Constanzo using FluCoMa Toolkit
    • ambisonic sound spatialization by ICST Institute for Computer Music and Sound Technology, which I play in quadraphonic set up if being supported by venues
    • mc. objects, which I can use to distribute single sound into multiple pitches and speeds at once
    • OSC data and Max communication
    • RNBO & Raspberry Pi for sound installation

    I'm interested in giving workshops/lectures/exchanges/residencies about programming in Max, spatial sound improvisation, arduino-Max incoporation, MIDI-Max control, and more. Please reach out to me (huichunyang1999@gmail.com). I'm happy to share and learn together!
